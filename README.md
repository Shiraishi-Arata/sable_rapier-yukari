# Building Sable's rapier for Android

该项目用于构建 Rapier 依赖库，使其能在 Android 端运行 [Sable](https://modrinth.com/mod/sable) 模组  
Sable 是 [机械动力：航空学（Create Aeronautics）](https://modrinth.com/mod/create-aeronautics) 的前置依赖  

This project is used to build the Rapier dependency library, enabling it to run the [Sable](https://modrinth.com/mod/sable) mod on Android.  
Sable is a required dependency for [Create Aeronautics](https://modrinth.com/mod/create-aeronautics).  

---
您可以前往 [Actions](https://github.com/MovTery/sable_rapier_android/actions) 获取构建的依赖库，自行修改模组本体  
也可以选择前往我们的原生库插件发布页，下载插件，这样就可以直接运行 Sable 模组：  

You can obtain the built dependency library from [Actions](https://github.com/MovTery/sable_rapier_android/actions) and modify the mod itself.  
Alternatively, you can visit our native library plugin release page to download the plugin, which allows you to run the Sable mod directly:   

https://github.com/ZalithLauncher/NativeLibPlugin/releases/tag/sable_rapier-0.1.0_NDK_BUILD
